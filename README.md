# plusコマンド
 * 標準入力から読み込んだ数字を足す

![test](https://github.com/NatsumaMatsunaga/robosys2022/actions/workflows/test.yml/badge.svg)

## 実行方法

* インストール

 `` $ git clone https://github.com/NatsumaMatsunaga/robosys2022.git ``

* 実行例

 `` $ seq 10 | ./plus ``

  ``55``

 ## 必要なソフトウェア
 * テスト済み: Python 3.7~3.10
 
 ## テスト環境
 * Ubuntu 20.04.5 LTS
